  • 1.使用构建工具vue-cli创建项目脚手架

    vue-cli是一个官方发布vue.js项目脚手架，使用vue-cli可以快速创建vue项目。 （1）安装 确认已安装了node.js，可在cmd中输入( node -v和npm -v），如显示出版号，说明安装成功。 通过vue ui安装 第一步：在cmd中输入vue ui 第二步：之后会弹出一个页面，点击创建 第三步：选择你项目要放的文件，然后点击在此创建新项目 第四步：输入项目名称，点击下一步 第五步：选择手动，点击下一步 第六步：根据项目需要勾选，本项目勾选了以下几个，勾选完后点击下一步 第七步：勾选router，本项目css选择stylus，然后点击创建项目 第八步：点击创建项目不保存预设 第九步：耐心等待一段时间 第十步：安装好后，进入到项目目录下，输入npm run serve运行，若出现最后一张图的页面，说明安装成功
